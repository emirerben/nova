template_matcher.match
Rule + AI scores

Greedy assignment using Gemini's energy/duration scores. Tight pass ±2s, loose pass ±6s, last-resort fallback. Per-clip usage cap ⌈N/K⌉ for variety. Pin clip[0] to slot[0] when it's a hook.

plan_slots — timing math
Rule

Snap each slot end to the nearest beat (±0.4s). Compute per-clip cursor so reuses don't repeat the same window.

collect_absolute_overlays — the rule-heavy stage
Rule

Skip "Welcome to PERU" combined overlays. Apply LABEL_CONFIG when content looks like a label. Sync font-cycle accel to curtain tail. Clamp end to slot end on curtain-close. Then merge same-text overlays across slot boundaries (gap < 2.0s) and truncate same-position overlaps.

Parallel slot rendering — reframe.py
Rule

Per slot: trim, scale, crop or pad to 1080×1920. HDR (HLG/PQ) tonemapped to SDR via zscale. Speed ramp via setpts. Curtain text pre-burned before bars close.

Insert interstitials
Rule

Curtain-close animation clamped to [4.0s, 60% of slot]. Short slots drop the curtain entirely.

FFmpeg concat / xfade joins
Rule

Translate Gemini's transition vocabulary (whip-pan, dissolve, hard-cut) to xfade types.

Burn overlays — text_overlay.py
Rule

Font-cycle = up to 60 PNGs per overlay. ASS subtitles for fade-in / typewriter / etc. Settle phase 30%, skipped when accel_at is set.

Mix audio, normalize, upload
Rule

Template audio mixed in. Normalized to -14 LUFS. Optional Gemini call for voiceover captions. Output uploaded to GCS.

→ job.status = "template_ready"

06Where the Just Fine bug stands

Gemini's job is "decide N slots." For Just Fine it decided N=1.

That dropped your second clip and ran short of audio. Two things you now have that fight that:

1
expand_slots — if K=2 clips and N=1, the orchestrator splits Just Fine's 1 slot at the nearest beat to make N=2 before matching. The rendered output now uses both your clips and fills the audio.
2
The divergence ⚠ in the editor — when the cached recipe's slot count differs from the rendered video's slot count, the editor shows it instead of silently lying.

Phase 2 (deeper fix) stops treating the recipe as fixed at admin time. The agent generates it at job time given (template constraints, K, beats). The K-vs-N negotiation becomes the agent's job, not consolidate/expand's.
