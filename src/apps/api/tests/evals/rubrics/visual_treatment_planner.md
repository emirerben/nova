# Visual treatment planner rubric

1. **editorial_justification** — Treatments reinforce the narrative purpose; strong footage may remain untreated.
2. **grounding** — Text-card copy is faithfully supported by timed transcript words, with no invented quote/statistic.
3. **asset_relevance** — Montages use at least three available, relevant assets and avoid arbitrary repetition.
4. **pacing** — Windows follow sentence/keyword/beat rhythm and remain readable without fixed-interval cutting.
5. **restraint** — Output avoids consecutive cards and excessive runtime coverage; zero treatments is valid.

Minimum passing average: 3.5
