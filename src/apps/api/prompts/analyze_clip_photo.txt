Analyze this still photograph for use in a short-form video (TikTok / Reels / Shorts).

You are a clip-discovery editor extracting the signal a downstream assembly agent needs to choose and order this photo within an edit. Return a JSON object with these exact fields:

- "transcript": string — always "" (photos have no spoken content).

- "hook_text": string — a single evocative phrase describing the photo's strongest visual hook for a short-form viewer (what makes them stop and look). Write as a short noun-phrase or observation, NOT a caption. Return "" if nothing stands out.

- "hook_score": float 0–10 — how strongly this image hooks the viewer if it appears as the first frame:
    • 0–2   generic / dull / no strong subject
    • 3–4   pleasant but expected
    • 5–6   interesting — holds attention for a beat
    • 7–8   strong — unusual moment, striking composition, or emotional resonance
    • 9–10  scroll-stopper — the image alone prompts curiosity or a reaction

- "detected_subject": string — short label naming the visible subject (e.g. "mountain peak at sunset", "person smiling at camera", "plate of food"). "" if no clear subject.

- "text_safe_zone": object {"x": float, "y": float, "w": float, "h": float} OR null — the largest calm rectangle where a big intro caption can sit without fighting the image (faces, the main subject, busy edges, burned-in text). Normalized 0.0–1.0 in the final 9:16 frame: x,y = top-left corner, w,h = width,height. Prefer open/calm areas (sky, a plain wall, blurred background, empty foreground). Return null if the frame is too busy for any confident safe rectangle. This field is REQUIRED — do not omit or return null without good reason, as intro text sizing depends on it.

- "visual_density": float 0–10 — how visually crowded the scene is. Anchors: 0–2 = sparse/empty (single subject, open background), 5–6 = normal, 9–10 = extreme clutter. This field is REQUIRED — do not default to 5 without judging the actual content.

- "composition_note": string — one short phrase on the frame's key compositional element (e.g. "subject centered, open sky above", "rule-of-thirds, subject left", "busy crowd fills frame"). "" if nothing notable.

- "content_type": one of "broll" | "action" | "ambience" — photos cannot be "talking_head" or have "voiceover":
    • broll — a subject or scene (the usual case for photos)
    • action — a frozen moment mid-motion (a jump, a splash, a kick)
    • ambience — pure atmosphere / landscape / mood with no clear action

- "audio_type": "ambient" — photos have no audio; always return "ambient".

- "best_moments": list with exactly ONE object — a still image covers a single hold window:
    { "start_s": 0.0, "end_s": 3.0, "energy": <float 0–10>, "description": "<one-line description of the photo's key visual>" }
    energy mirrors hook_score (same 0–10 scale). Do NOT return an empty list.

Return ONLY the JSON object. No markdown fences, no explanation outside the JSON.
