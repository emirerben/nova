Shot-cut detection — a new slot starts when the image changes via:
- Hard cut
- Whip pan (the post-whip shot is a new slot)
- Speed ramp that breaks shot continuity (use speed_factor on the affected slot)
- Zoom-in cut or other intentional intra-shot break
- Curtain-close interstitial (the bars themselves are an interstitial, NOT a slot — the slot after the curtain is a new slot)
- Flash to black/white (the post-flash shot is a new slot)

A new slot does NOT start for:
- Cross-dissolves within a single thematic shot — treat as one slot, set transition_in="dissolve" on the next true cut
- Camera moves (pans, tilts, zooms) without a cut — record in camera_movement on the existing slot
- Audio-only overlaps (J-cut / L-cut) — visual stays one slot

Hard rules:
- slots array length MUST equal shot_count
- The sum of all target_duration_s values must approximately equal total_duration_s (±0.05s tolerance, ±5s logged as warning)
- position is 1-indexed temporal order (slot 1 = first shot)
- EVERY slot object MUST include an 'energy' field (float 0–10) reflecting the combined musical + visual intensity at that slot — listen to the audio track AND watch the cut frequency. Anchor to:

Black-segment classification. $black_segments_context above lists detected dark frames between shots. For each black segment, observe how darkness arrives and exits and record an entry in the interstitials array:
- Bars close from top+bottom, then reopen   →  type="curtain-close", set transition_in="curtain-close" on the NEXT slot
- Gradual fade in, sustained hold, fade up  →  type="fade-black-hold"
- Bright overexposure flash, short hold     →  type="flash-white"
- Hard cut with a few frames of black hold  →  type="fade-black-hold" with short hold_s

Frame count matters in interstitials: 2–4 frames is rhythmic emphasis; 12–24 frames is a section break; 1s+ is a chapter close. Set animate_s and hold_s accordingly.
