GUIDELINES
----------
1. Pick the set whose tags + vibe best match the creator's content tone and audience.
   Travel/cinematic/outdoor → editorial sets. Lifestyle/wellness → clean minimal.
   Bold/statement/motivational → word_reveal or bold sets. Fashion/beauty → high_fashion.
   Authentic/personal → film/mono. Versatile/mixed → default or lifestyle_clean.

2. Knob overrides: apply ONLY when the persona or TikTok data clearly supports it.
   - font_family: override only to shift tone (e.g. a serif for a more editorial feel
     when the creator posts aesthetics; sans for a more casual/accessible feel).
     Editorial serifs + smaller sizes → tasteful. Loud sans at max size → cheap-looking.
     PREFER no override (null) unless there is a strong persona signal.
   - text_size_px: override only when the creator's footage type demands it. Dense
     urban footage → smaller (~42-50). Open nature/architecture → can be larger (~60-70).
     Range: 40–80 px. PREFER null (let the auto-sizing compute from footage).
   - position: override only when the creator's content type has a pattern (e.g.
     talking_head creators → "bottom" or "top" avoids covering their face). Otherwise null.
   - text_color, highlight_color: override only for brand consistency
     (e.g. a creator known for a specific color). Format: "#RRGGBB". PREFER null.
   - text_anchor: "left" only for text-heavy creators; "center" is the default. PREFER null.
   - stroke_width, cycle_fonts, position_x_frac, position_y_frac: PREFER null.
   Do NOT emit any key not listed in the schema (strict validation in the API).

3. footage_type_bias: list the clip types this creator's content mainly uses, from:
   ["talking_head", "broll", "action", "ambience"]. Order most → least common.
   E.g. a vlogger → ["talking_head", "broll"]. A nature creator → ["broll", "ambience"].

4. preferred_edit_format_mix: weights summing to ≤ 1.0 for each format the creator
   should use. Only include formats clearly supported by their persona/TikTok data.
   Formats: "montage", "talking_head", "day_vlog", "single_hero". E.g.:
   {"montage": 0.5, "talking_head": 0.3, "day_vlog": 0.2}

5. instruction_level: how much filming guidance this creator wants.
   "full" = they want detailed shot lists. "light" = high-level suggestions only.
   "none" = they upload whatever they want, no filming instructions.
   Default to "full" unless the persona explicitly signals otherwise.

6. rationale: one or two sentences explaining WHY you chose this set and any overrides.
   Be specific to this creator's persona and TikTok data.
